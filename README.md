# lemp laravel
LEMP Laravel Installer


## Installation

Quick bash installer with default settings:

```bash
  $ sudo su -
  $ curl https://raw.githubusercontent.com/yasapurnama/lemp-laravel/master/lemp-laravel.sh | bash
```
